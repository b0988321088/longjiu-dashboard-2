# 龍九控股戰術儀表板 — 視覺化線框圖（Wireframes）
設計原則：F型掃描、5秒異常識別、倒金字塔、漸進式揭露

---

## 設計系統規範

### 色彩語意
| 色彩 | 語意 | 使用場景 |
|------|------|----------|
| 🟦 藍色 #3B82F6 | 穩定監控中 | 正常數據、銀行帳戶、現金流 |
| 🟧 橙色 #F59E0B | 需調度/觀察 | 星展補庫、溢價偏高觀察 |
| 🟥 紅色 #EF4444 | 需立即處理 | 0056嚴重溢價、0056質押、貝萊德溢價 |
| 🟪 紫色 #8B5CF6 | 戰略決策 | 巴菲特視角、裁決按鈕 |
| 🟩 綠色 #10B981 | 充裕/正向 | Runway充足、盈餘正值 |
| ⬛ 灰色 #6B7280 | 靜態資訊 | 行事曆、固定排程 |

### 卡片系統
- **KPI 卡片**：左上 3-5 個，大字數字，邊框顏色對應状态
- **圖表卡片**：圓角矩形，內含圖表視圖
- **警示卡片**：紅/橙色左邊框，內文對應
- **明細卡片**：可折疊，點擊展開

### 響應式適配
- **桌面端**：4 欄網格，左側 3 欄主內容 + 右側 1 欄警示
- **平板端**：2 欄網格
- **手機端**：單欄，僅顯示 KPI + 異常警示，明細隱藏
