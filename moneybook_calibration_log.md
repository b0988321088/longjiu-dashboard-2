## 計算方法備忘

```
銀行資產(正數) = Σ(帳戶金額) where 帳戶金額 > 0，負數帳戶不列入資產
真實月支出 = 近30天明細中，排除：帳戶互轉、貸款本金、投資買賣、繳信用卡費、外幣買賣
            保留：飲食、住家、購物、交通、休閒、學習、服飾、費用/手續費、現金消費
信用卡本月待繳 = 帳單表中，取最近一期（依繳費截止日最近的未過期帳單）
```
