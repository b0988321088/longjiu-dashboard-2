## 四、每日更新流程（最小變更版）

### Step 1：昨日快照 vs 今日真值
```bash
python daily_diff_check.py \
  --old 昨日_snapshot.json \
  --new 今日_snapshot.json
```
- 有變化 → 記下哪些 key 要改
- 無變化 → 不動檔案，流程結束

### Step 2：只改變化字段
| 變化來源 | 修改目標 |
|---------|---------|
| Moneybook CSV 更新 | `monthly_income` / `total_expense` |
| 今天有配息入帳 | `insurance_monthly_dividend` page1 income |
| 今天有支出 | `total_expense` 明細 |
| 行程變動 | calendar 區塊 |
| 資產市值變動 | `total_assets` / `securities_total_market_value` |
| 房租入帳 | `rent_collected` / `moneybook_cash` |

### Step 3：Preflight 掃描
```bash
# 掃描所有前端檔案
python .../preflight_check.py --target index.html

# snapshot 特殊檢查
python .../preflight_check.py --target dashboard.py
```
- PASS → 繼續
- FAIL → 修完再過

### Step 4：CIO 審查
- 數據：top-level vs page1 vs page4 是否一致
- 程式：語法通過 + 無格式字串 crash
- 數字：沒有舊數字殘留

### Step 5：上傳 + curl 驗證
```bash

# 驗證線上生效
```
